¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Diferencias:

  • Java es un lenguaje de programación de propósito general, mientras que JavaScript es un lenguaje de programación de scripts utilizado principalmente para crear páginas web interactivas.
  • Java se ejecuta en una máquina virtual mientras que JavaScript se ejecuta directamente en el navegador web.
  • Java es un lenguaje de programación tipado, mientras que JavaScript es un lenguaje de programación no tipado.
  • Java utiliza una sintaxis similar a la de C++ mientras que JavaScript utiliza una sintaxis similar a la de ECMAScript.

Semejanzas:

  • Ambos lenguajes son orientados a objetos y soportan herencia y polimorfismo.
  • Comparten muchas estructuras de control y operadores.
  • Ambos tienen una amplia variedad de bibliotecas y frameworks disponibles para su uso.

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

  • getFullYear(): Devuelve el año de una fecha como un número de cuatro dígitos
  • getMonth(): Devuelve el mes de una fecha como un número (0-11)
  • getDate(): Devuelve el día del mes de una fecha como un número (1-31)
  • getHours(): Devuelve la hora de una fecha como un número (0-23)
  • getTime(): Devuelve el tiempo transcurrido en milisegundos desde el 1 de enero de 1970 hasta la fecha dada

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

  • push() - agrega uno o más elementos al final de un arreglo y devuelve la nueva longitud del arreglo
  • pop() - elimina el último elemento de un arreglo y lo devuelve
  • slice() - devuelve una copia de una porción del arreglo, especificada por un índice de inicio y un índice final
  • indexOf() - devuelve el índice de la primera ocurrencia de un elemento en un arreglo, o -1 si no se encuentra
  • forEach() - ejecuta una función para cada elemento en un arreglo

Puede causar ciertos problemas de depuración en el código, ya que de esta forma no se puede estar seguro a primera instancia el lugar en donde se cambia el valor de la variable global, o en donde está causando problemas, si se usa en distintos lados (como lo es en las funciones).
